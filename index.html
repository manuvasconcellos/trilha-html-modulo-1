Trabalhando com mídias

Tags como <img>, <audio>, <video> e <iframe> permitem inserir diferentes tipos de mídia em uma página.

Tabelas em HTML

Para criar tabelas, usamos as tags <table>, <tr>, <td> e <th>. Elas ajudam a organizar dados tabulares.

HTML Semântico

HTML semântico usa tags como <header>, <main>, <section>, <article> e <footer> para dar significado ao conteúdo.
